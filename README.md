# 1. Blinkit Sales Analysis Dashboard (MS Excel)

## 2. Project Overview

This project involves a comprehensive analysis of sales data for **Blinkit**, India's last-minute app. The goal was to extract actionable insights into sales performance, customer preferences, and outlet efficiency using **MS Excel**. I developed an interactive dashboard to visualize key metrics, enabling data-driven decision-making regarding inventory and outlet management.

## 3. Key Insights & Features

* **KPI Tracking:** Monitored essential metrics including **Total Sales ($1.20M)**, **Average Sales ($14)**, **Number of Items Sold (8523)**, and **Average Customer Rating (3.97)**.
* **Sales by Item Type:** Identified top-performing categories, with **Fruits and Vegetables ($178.1K)** and **Snack Foods ($175.4K)** leading in revenue.
* **Fat Content Analysis:** Analyzed sales distribution based on product health attributes, revealing that **Low Fat** items account for **65%** of total sales compared to 35% for Regular items.
* **Outlet Performance:** Compared sales across different outlet types and sizes, noting that **Supermarket Type 1** is the highest contributor with **$787.5K** in sales.
* **Geographic Distribution:** Visualized sales performance across different city tiers, with **Tier 3** locations generating the highest revenue at **$472.1K**.
* **Establishment Trends:** Evaluated the growth of sales from 2011 to 2022, identifying a peak in **2018 ($204.5K)**.

## 4. Technical Skills Applied

* **Data Cleaning:** Handled raw dataset inconsistencies to ensure accurate reporting.
* **Data Modeling:** Used **Pivot Tables** to aggregate and summarize large volumes of sales data.
* **Advanced Formulae:** Leveraged Excel functions for calculating dynamic KPIs.
* **Data Visualization:** Designed an interactive interface using various charts (Donut, Bar, Area, and Funnel charts) and **Slicers** for real-time filtering by Outlet Size, Location Type, and Item Type.
